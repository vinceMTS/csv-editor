# CsvEditor.WinUI prototype

This is a minimal WinUI 3 prototype for the CSV Editor.

Prerequisites:
- Windows 10 (19041+) or Windows 11
- .NET SDK 7.0
- Windows App SDK (installed via Visual Studio or the NuGet package)

Build and run:

```
dotnet build src/CsvEditor.WinUI/CsvEditor.WinUI.csproj
dotnet run --project src/CsvEditor.WinUI/CsvEditor.WinUI.csproj
```

Notes: This is an initial scaffold; open the project in Visual Studio 2022/2023 for designer support and the Windows App SDK workload.
